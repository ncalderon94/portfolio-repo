Project: Customer Service.

Overview

This scenario-based eLearning project was developed for Customer Service Representative personnel for companies. The goal with this scenario-based training is to help the customer support staff streamline the approach to better assist via email with any problem the client is facing while enabling satisfactory interactions. This learning solution would put the customer service representative in a realistic simulation with a client.

  • Audience: Customer Service Representatives.
  • Tools used: Microsoft PowerPoint, Procreate, Articulate Storyline 360, Google Docs, GitMind.
  • Responsibilities: Instructional Design, eLearning Development, Graphic Design.

Process

I designed this training by defining the company goal in an action map, redacting the text-based Storyboard with its different outcomes, creating a visual mockup, and completing the development of this project.

Design Theory

With the help of the SME, I created an Action Map to identify the tasks the customer support staff needed to perform in order to be aligned with the company’s training goal. After setting 5 main tasks, we expanded them using branches. By breaking the main tasks down, we established a basic blueprint on how to create scenario-based learning.

Text-Based Storyboard

I used Google Docs for the text-based storyboard.

Visual Mockups

The visual aspect of a project is always my favorite part of the process. I was able to design a learning experience that was aesthetically pleasing as well as functional for the user. I found some visuals and used them to create the mock-ups. For the colors, I decided to maintain uniformity by keeping the tones pleasant for the eye and simultaneously, non-distracting for the user. Tool used for the color scheme was Coolors.
