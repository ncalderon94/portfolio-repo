Project: Identify Phishing Emails

Experience the project

Overview

In this project, the user will be able to review the most common characteristics of a phishing email in order to, in the future, identify phishing emails with ease.

  • Audience: Computer users.
  • Tools used: Microsoft PowerPoint, Articulate Storyline 360 and Google Docs.
  • Responsibilities: Instructional Design, eLearning Development, Graphic Design.

Process

I designed this project by meeting with a SME to identify the characteristics of a phishing email. After gathering the information, I redacted the text-based learning. Then, I created a visual mockup and finalized the development of this project.

Design Theory

With the help of the SME, I created a table identifying the different aspects of a phishing email. We also discussed different consequences of failing to identify phishing emails, as well as the recurrence of the manner.

Visual Mockups

I chose to use a set of subtle tones of colors on the mockup to maintain simplicity throughout the project.

Project: Create your ID

Experience the project

Overview

The purpose of this project was to create multiple variables, triggers and sets of rules to get the desired outcome using Articulate Storyline 360. In this project, the user will create a job ID by selecting a character, entering their name, job title, and finally selecting a face expression for the ID. By the end of this activity, the user would have created a job ID with their name and job title.

  • Audience: Everyone
  • Tools used: Microsoft PowerPoint and Articulate Storyline 360.
  • Responsibilities: Development and Graphic Design.
